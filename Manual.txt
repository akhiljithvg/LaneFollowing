------------------------------------------------------------
IMPORTANT NOTES
------------------------------------------------------------

1. RPi.GPIO works ONLY on real Raspberry Pi hardware.
2. opencv-python already includes ArUco support.
3. picamera2 is optional:
   - Required for CSI camera
   - Not required for USB camera
4. Do NOT install opencv-contrib-python on Raspberry Pi.
5. Run the main script with sudo if GPIO access is denied:

   sudo venv/bin/python3 lane_follower.py
